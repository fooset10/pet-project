## 🎮 Об игре

«Футбольный пенальти» — это аркадная игра, в которой вы выступаете в роли исполнителя пенальти.  
Ваша задача — забить как можно больше голов вратарю, который умеет прыгать в разные стороны и иногда угадывает ваши намерения.

Игра сочетает в себе:
- Элементы случайности (вратарь выбирает сторону случайно)
- Реалистичную физику полёта мяча
- Динамичную анимацию вратаря и мяча
- Систему сэйвов и голов

---

## 🧠 Как играть?

У вас есть **10 попыток**.  
Вы выбираете направление удара:

| Клавиша | Направление |
|---------|-------------|
| `1`     | Левый угол  |
| `2`     | Центр       |
| `3`     | Правый угол |

Вратарь:
- В 40% случаев прыгает влево
- В 40% — вправо
- В 20% — остаётся в центре

Ваш гол засчитывается, если:
- Мяч попадает в створ ворот **И**
- Вратарь не смог его поймать

---

## 🧩 Механики

### 🧤 Вратарь
- Анимированный прыжок в сторону
- Разная вероятность сэйва в зависимости от:
  - Совпадения направления удара и прыжка
  - Расстояния до мяча
  - Случайного фактора

### ⚽ Мяч
- Летит по траектории к выбранному углу
- При сэйве меняет цвет (оранжевый)
- При попадании в створ — засчитывается гол

### 📊 Интерфейс
- Счёт голов
- Количество сэйвов вратаря
- Индикатор прогресса
- Подсказки и анимация результата
- Оценка игры после 10 ударов

## 🚀 Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/yourusername/football-penalty.git
cd football-penalty
```

2. Установите зависимости
```bash
pip install pygame
```
3. Запустите игру
```bash
python penalty_game.py
```



🧪 Требования
Python 3.7+
Pygame 2.5+




🎯 Советы для победы
Бейте точно в угол — это снижает шанс сэйва

Не бойтесь бить в центр, если вратарь часто прыгает

Следите за статистикой: если вратарь часто ловит справа — пробуйте слева

Идеальный результат — 10 из 10 голов 🏆
